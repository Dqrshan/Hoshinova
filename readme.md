## Hoshinova - The Bot
The bot's framework is inspired by [reconlx](https://youtube.com/reconlxx). The bot uses `discord.js` v13 with **slash commands** support!
Features
* Owner-only commands (Eval, Reload, etc.)
* Moderation, Utility, Information commands.
